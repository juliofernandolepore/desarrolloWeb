declaracion de variables

existen 2 formas de declarar variables, la forma simplificada que autodetecta el tipo de datos, nombre de la variable seguido de := y el valor de la literal, se pueden declarar multiples variables en una sola linea. La otra declaracion de variable es propia de la sintaxis de c, var variable1 string = "hola", aun asi puede declarar con valor null. La ventaja de establecer el tipo de dato que contendra la variable, previene bugs, errores de compilacion y un codigo mas legible.

estructura switch

la estructura switch es utilizada en multiples lenguajes, simplifica una sintaxis abultada de condicionadas if-else anidadas, siempre siendo necesario el uso de break para salir de la iteracion.

Struct (Estructuras)

Struct nos sirve para definir tipos de datos mas precisamente una coleccion, se asemeja a las Clases en otros lenguajes (POO), nos ahorra tiempos de escrituras, mejora la calidad del codigo y evita errores.

Punteros en Go (pointers)

Los punteros son apuntadores que su funcion en este como otros lenguajes como C o C++ que la memoria sea optimizada al ser referenciadas a un mismo punto de memoria ram para distintas variables o funciones por ejemplo

Iteraciones con el metodo Range

El metodo range permite recorrer un arreglo o coleccion proporcionamos "clave y valor", y retorna indices con los valores que requerimos. Es un metodo muy util que no descarta el uso de iterador for.
